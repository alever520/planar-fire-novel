## 🎯 核心设定

### 力量体系（对齐要点2：成长曲线）

**法师等级**（D&D改编）
```
见习法师 (LV1-3)   → 基础魔法，学院学生
正式法师 (LV4-6)   → 掌握2环法术，可独立执行任务
资深法师 (LV7-10)  → 掌握4环法术，城镇保卫者
高阶法师 (LV11-14) → 掌握6环法术，领主级战力
大法师 (LV15-18)   → 掌握8环法术，国家级战力
传奇法师 (LV19-20) → 掌握9环法术，位面级战力
神话级 (LV21+)     → 半神领域
```

**艾伦成长路线**
- 第1章：LV5 战斗法师（导师刚死，接受符文）
- 第10章：LV7（击败教团分部长，觉醒火元素掌控）
- 第30章：LV10（击败五元素教派首领，觉醒五元素融合）
- 第70章：LV14（深渊战争胜利，掌握位面之力）
- 第130章：LV18（击败龙王萨洛克斯，成为传奇守护者）
- 第200章：LV20+（最终战）

**符文之力进阶**
```
阶段1 (1-10章)   → 单元素（火焰）
阶段2 (11-30章)  → 双元素融合（火+雷）
阶段3 (31-70章)  → 五元素平衡
阶段4 (71-130章) → 位面之力觉醒
阶段5 (131-200章)→ 神话守护者
```

## 🎭 角色设计（对齐要点4、5）

### 主角：艾伦·科尔温
**初始状态**（第1章）
- 等级：LV5 战斗法师
- 性格：理性、谨慎、重情义
- 缺陷：过于依赖逻辑，不擅长政治手腕
- 目标：查明导师死亡真相

**成长弧线**
- 前期：复仇者（被仇恨驱动）
- 中期：守护者（理解责任）
- 后期：领袖（平衡各方利益）
- 终期：哲学家（超越善恶对立）

**核心矛盾**
- 复仇冲动 vs 理性判断
- 个人自由 vs 守护者责任
- 力量诱惑 vs 人性坚守

### 反派设计（对齐要点4：立体反派）

#### 伊格纳修斯（永恒之炎教团领袖）
**身份**：塞尔修斯（导师）的师兄，前守护者候选人

**动机（合理化！）**
- **信念**："世界正在毁灭（魔能污染、位面崩塌），唯有释放元素原初之力，重塑世界，才能拯救文明。"
- **悲剧性**：他真的相信自己在"拯救世界"
- **过往**：50年前目睹家乡因魔能污染毁灭，无力阻止
- **分歧点**：塞尔修斯认为"守护现状，渐进改良"，伊格纳修斯认为"毁灭重生"

**智商在线**
- 不会"明知艾伦厉害还要装逼"
- 每次行动都有充分准备和后手
- 能预判艾伦的行动，设下陷阱
- 愿意战术撤退，不送人头

**魅力值**
- 有个人魅力，吸引大量追随者
- 对部下真诚（不是恶魔般使用工具人）
- 与艾伦对话时，能说出让人动摇的理念

**阶段性失败合理化**
- 第10章：并非本体出现，只是分部长被击败
- 第30章：战术性撤退，达成"唤醒元素位面"的部分目标
- 第70章：主动释放深渊恶魔，制造混乱，实则在寻找"创世之炎"
- 第130章：与艾伦达成短暂合作（对抗虚空），展现复杂性
- 终章：最终战，理念冲突的终结

---

#### 阶段性反派（避免重复）

**第一卷 (1-10章)** - 狂信徒型反派（蠢货型，快节奏消耗）
- 教团火焰教区长官：莫里斯·炎心
- 特点：狂热、自大、送人头型反派
- 作用：给艾伦练手，展现战斗力

**第二卷 (11-30章)** - 阴谋家型反派（智商在线）
- 五元素教派首领：塞拉·暴风
- 特点：工于心计，善于伪装，能逃脱
- 作用：让艾伦学会"不是所有敌人都能一次消灭"

**第三卷 (31-70章)** - 强敌型反派（实力碾压）
- 深渊领主：巴洛克
- 特点：纯粹的力量压制，正面交锋艾伦根本打不过
- 作用：倒逼艾伦突破，学会"不是所有战斗都靠硬打"

**第四卷 (71-130章)** - 悲剧型反派（让读者共情）
- 红龙长老萨洛克斯
- 特点：被虚空腐化，本质是受害者
- 作用：让艾伦思考"敌人不一定是恶人，战争不一定有对错"

### 第1章：灰烬中的觉醒（4500字）
**对齐要点1：开篇抓人，前三章定生死**

#### 开场设计（直接从冲突切入！）
**❌ 错误开场**："艾伦·科尔温是一名普通的法师学徒..."（无聊）
**✅ 正确开场**："爆炸撕裂了夜空。"（震撼）

#### 章节结构

**第一幕：爆炸（1000字）**
- 开场：魔能反应堆突然爆炸，火焰冲天
- 艾伦正在实验室，被冲击波掀翻
- 听到导师在核心区域，不顾危险冲进火海
- **小爽点**：展现战斗法师的实力（法盾抗伤害、疾风步快速移动）
- **钩子**：在火光中看到一个戴兜帽的人逃离，胸口有火焰徽记

**第二幕：导师之死（1500字）**
- 在废墟中找到导师，全身烧伤
- 导师临终对话（简洁有力！）：
  - "艾伦...对不起...我本该早点告诉你..."
  - "守护者的...责任...比我想的更重..."
  - "他...选择了错误的道路...阻止他..."
  - "符文...它会选择你...（掏出符文石）承受痛苦...守护..."
  - 导师死去，符文石碎裂
- **符文觉醒场景**（视觉化）：
  - 金红色能量涌入身体
  - 剧烈疼痛，像被烈火焚身
  - 皮肤浮现复杂的符文纹路，发光发热
  - 幻象：看到无数守护者的记忆碎片（快速闪过，不解释）
  - 昏迷前听到一个声音："继承者...守护位面...阻止火焰..."
- **情感高潮**：艾伦悲痛欲绝，抱着导师遗体

**第三幕：成为嫌疑人（1200字）**
- 醒来在学院医疗室，身体被束缚
- 格雷森教授（导师挚友）表情复杂地告知：
  - "艾伦，学院认为你涉嫌参与反应堆爆炸案..."
  - "证据是...你的魔力残留在爆炸核心..."
  - "还有...导师最近的研究笔记里，提到了禁忌魔法..."
- 艾伦愤怒反驳："我在救人！你知道导师不会做这种事！"
- 格雷森低声："我相信你...但联邦议会不这么想...他们要找替罪羊..."
- 艾伦发现自己的法术权限被封印，被软禁在宿舍

**第四幕：发现真相线索（800字）**
- 独自一人时,艾伦整理导师遗物
- 发现导师留下的隐秘笔记（触发符文之力才能显现）
- 笔记内容（部分）：
  - "永恒之炎教团...计划启动元素召唤仪式..."
  - "伊格纳修斯...我的师兄...他疯了...他要毁灭世界来拯救世界..."
  - "守护者符文...只能给予艾伦...他比我更适合..."
  - "如果我死了...艾伦会被针对...必须尽快逃出学院..."
- **伏笔埋下**：
  - 笔记提到"创世之炎"（长线伏笔）
  - 提到"五大元素位面的平衡正在崩溃"（中线伏笔）
  - 提到"伊格纳修斯曾是守护者候选人"（长线伏笔）

**第五幕：决定逃脱（1000字）**
- 艾伦意识到：留在学院=坐以待毙
- 符文之力再次觉醒（感应到危险）
- 夜晚，艾伦使用符文之力（火焰操控）烧毁封印
- **展现代价**：使用符文后身体剧痛，符文纹路发红发烫
- 潜行出学院，被巡逻的魔像发现
- **小爽点**：展现符文之力的强大（一发火焰箭击毁魔像）
- 逃到城墙边，回望学院，内心独白：
  - "导师...我会查明真相...我会阻止他们..."

**章节结尾钩子**（对齐要点1）
- 逃出学院时，艾伦感应到一股强大的火元素波动
- 方向是...城外的熔岩峡谷
- "教团的仪式...已经开始了？"

**章节爽点统计**
- 小爽点1：冲进火海救导师（勇气+实力展现）
- 小爽点2：符文觉醒（获得上古力量）
- 小爽点3：烧毁封印逃出学院（反抗体制）

**伏笔统计**
- 短线伏笔：火焰徽记的人是谁？（第3章回收）
- 中线伏笔：教团的仪式目的？（第10章回收）
- 长线伏笔：伊格纳修斯的真实身份（第30章回收）
- 长线伏笔：创世之炎是什么？（全书最终回收）

### 第2章：暗影刺客（4000字）
**目标**：引入第一个伙伴莉拉，展现艾伦的战斗智慧

#### 核心冲突
艾伦 vs 教团刺客 + 被追杀的半精灵刺客（莉拉）

#### 剧情要点
- 艾伦逃到贫民区，遇到被教团追杀的莉拉
- 教团刺客：3名火焰刺客（LV6），目标是夺取莉拉盗取的教团情报
- 艾伦误打误撞卷入战斗
- **打脸爽点**：教团刺客轻视艾伦（"一个学院跑路的学生"）→被艾伦吊打
- **战斗展现**：
  - 艾伦的战术智慧（利用环境、陷阱、魔法组合）
  - 符文之力的强大（火焰掌控超越常规法师）
  - 与莉拉的配合（虽然是第一次，但战术契合度高）
- 战斗结束，莉拉重伤，艾伦救她
- 莉拉：" 你为什么救我？"
- 艾伦："我也在被追杀...敌人的敌人就是朋友。"
- **伏笔**：莉拉盗取的情报提到"熔岩峡谷的火焰祭坛"

**章节结尾钩子**
- 两人决定合作，前往熔岩峡谷破坏教团仪式
- 莉拉冷冷地说："你知道教团有多强吗？去了可能会死。"
- 艾伦："那也要去。我要为导师报仇。"
- 莉拉沉默片刻："...我也有要报的仇。"
